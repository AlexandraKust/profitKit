# ProfitKit
Тестовое задание компании ProfitKit

Выполнено:
+ с помощью препроцессора SCSS
+ по методологии БЭМ
+ кроссбраузерно, адаптивно, по семантике 
+ pixel-perfect
+ ванильный JS + swiper
+ собран и оптимизирован на gulp
